A linked list is a linear data structure made up of individual elements called nodes, where each node contains data and a reference (or pointer) to the next node in the sequence. Unlike arrays, linked lists do not store elements in contiguous memory locations; instead, each node can be stored anywhere in memory, and the connection between nodes is maintained through pointers. The first node is called the head, and the last node typically points to null to indicate the end of the list. Because elements are connected by references rather than position, linked lists allow efficient insertions and deletions without shifting other elements, especially when modifying the beginning or middle of the list. However, accessing a specific element requires traversing from the head node step by step, making random access slower compared to arrays. Variations include singly linked lists, where each node points only to the next node, doubly linked lists, where nodes point to both the next and previous nodes, and circular linked lists, where the last node points back to the head.
